# Smart Complaint Management System

AI-Based Smart Complaint Management System using MERN Stack.

## Features

- User Signup & Login
- JWT Authentication
- Complaint Registration
- View Complaints
- Update Complaint Status
- Delete Complaint
- Search Complaint by Location
- AI Complaint Analysis
- MongoDB Integration

## Technologies Used

- MongoDB
- Express.js
- Node.js
- JWT
- bcryptjs

## Installation

```bash
cd backend
npm install
npm run dev
